An interactive Streamlit app to compare performance metrics between two companies using uploaded CSV files.

## Features

- Display total revenue KPIs
- Interactive bar chart (Revenue or Units Sold)
- Expandable data preview

## Data Format
Each CSV should include:

Jan,10000,500

...


How to Run
Install dependencies:
               pip install streamlit pandas plotly
Run the app:
               streamlit run app.py
